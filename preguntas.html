Instrucciones para descargar preguntas

  • Las preguntas se guardan automáticamente en un archivo Excel con nombre aleatorio
  • Cada pregunta nueva se agrega al archivo existente
  • Para descargar todas las preguntas, haz clic en el botón "Descargar Excel"
  • El archivo se descargará con un nombre encriptado para mayor seguridad
Nota de Seguridad: El archivo Excel se genera con un nombre aleatorio (ej: "datos_X8f3zQ.xlsx") para evitar que los usuarios lo encuentren. Solo tú puedes acceder a través de este panel con la contraseña correcta.

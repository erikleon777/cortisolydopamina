Funcionamiento del Sistema

  • Cada pregunta se guarda automáticamente en un archivo Excel
  • El archivo tiene un nombre aleatorio para mayor seguridad
  • Los usuarios NO pueden ver las preguntas de otros
  • Para descargar todas las preguntas, usa el botón "Descargar Excel"
Nota de Seguridad: El archivo Excel se genera con un nombre aleatorio (ej: "preguntas_x7f9q2.xlsx") para evitar que los usuarios lo encuentren. Solo tú puedes acceder a través de este panel con la contraseña correcta.
